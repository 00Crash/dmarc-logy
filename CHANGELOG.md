# Changelog

## v1.0

Finální produkční balík po testování.

- Stabilizované UI pro 1920 x 1080.
- Home stránka: přehledy, zdroje odesílání, doporučení.
- Historie: vývoj v čase a importované reporty.
- Přihlášení přes `.env`.
- Zdroje odesílání po kombinaci IP + Header From doména.
- Filtry Header From a Stav zdroje.
- Defaultní filtr Stav zdroje = neznámý.
- Kompaktní detail zdroje v jednom řádku.
- Čitelnější dlouhé IPv6 zdroje a provider hodnoty.
- Doporučení jako kompaktní seznam úkolů.
- Vývoj v čase sestupně podle data.
- Importované reporty sestupně podle data importu a období reportu.
- Rozšířený sloupec soubor.
- Produkční dokumentace pro Ubuntu Server 24.04.4.
- DMARC rollout dokumentace.
- Provozní a bezpečnostní dokumentace.
- Docker log rotation.
